# APK Dialog Injector

כלי אוטומטי שמקבל קובץ APK, מוסיף לתוכו דיאלוג מותאם אישית בעברית עם כפתורים:
- **אישור**
- **אל תציג שוב** (CheckBox)
- **ביטול**

הדיאלוג יופיע בפעם הראשונה שהאפליקציה נפתחת, ואם המשתמש מסמן "אל תציג שוב" ולוחץ "אישור", הוא לא יופיע שוב.

## איך משתמשים?
1. צור **Issue** חדש ב‑GitHub עם התווית `patch-request`.
2. העלה את קובץ ה‑APK (או ספק קישור) ופרט את הכותרת, התיאור וקישור התמונה (אופציונלי) בתוכן ה‑Issue.
3. ה‑GitHub Action ירוץ אוטומטית, ייצור גרסה מתוקנת של ה‑APK ויחזיר אותה כתגובה ב‑Issue.

## דרישות
- חשבון GitHub עם הרשאה ל‑Actions (בפרויקט ציבורי זה חינמי).
- אין צורך בהתקנות מקומיות – הכל רץ בענן.

## פיתוח
הפרויקט בנוי מ‑Docker image שמכיל `apktool`, `jarsigner`, וסקריפט Python שמבצע את ההזרקה.  
ה‑GitHub Action מפעיל את ה‑Docker container עם פרמטרים מה‑Issue.
